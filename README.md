# 💌 Do You Love Me?

A beautiful, interactive digital love confession website featuring glassmorphism UI, smooth animations, and playful logic.

## ✨ Key Features
- **Interactive Confession:** Multi-screen flow with a "Fleeing No" button and "Growing Yes" button.
- **Visual Effects:** Parallax floating hearts, confetti explosions, and magical sparkles.
- **Modern UI:** Responsive glassmorphism design using Tailwind CSS.
- **Audio:** Integrated background music with a toggle control.

## 🚀 Quick Start
1. **Clone** the repository.
2. **Open** `index.html` in any modern browser.
*No build process or installation required.*
